# HMPT
Hi, this repository contains the code and the data for the TKDD paper "Cross-Modal Multiple Granularity Interactive Fusion
Network for Long Document Classification"

To download the dataset for "MAAPD", "Review" and "Food101", pls go to this URL:  
https://drive.google.com/drive/folders/1759nBGt7J0ZkUK8-jSbAys9_SANQVC-E?usp=drive_link

For the dataset of MMaterials, due to copyright restrictions, it is not convenient to disclose directly. If needed, please feel free to contact me via email for access tfliu@emails.bjut.edu.cn.

If you use the MMaterials and MAAPD datasets in your research, please cite:
